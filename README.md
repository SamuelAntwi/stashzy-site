# stashzy-site
Stashzy is a privacy-first home inventory app that helps you catalog items with photos, track quantities and replacement timelines, and get smart reminders so you stop wasting money on duplicates and expired supplies.

Key features
	•	Photo-based item capture (fast cataloging)
	•	Categories + locations (room-by-room organization)
	•	Quantity tracking with min/max alerts
	•	Replacement planning (manual + smart suggestions in later versions)
	•	Search, filters, and exports (CSV / insurance-ready outputs in later versions)
	•	Offline-first with optional iCloud backup

Why Stashzy

Most inventory apps feel like business tools. Stashzy is built for real households: simple to use, fast to update, and focused on saving time and money.
